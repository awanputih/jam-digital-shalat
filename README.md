# ⏰ Jam Digital Shalat - ESP8266 + Android

Aplikasi lengkap untuk menampilkan jadwal shalat dengan multi-metode hisab yang akurat.

## Features:
- Real Time Clock
- Jadwal Shalat Offline
- Display LCD/OLED
- Azan & Reminder
- Kontrol via Android
- Non-Volatile Storage
- Multi-City Support

## Hardware:
- ESP8266
- RTC DS3231
- LCD/OLED
- Buzzer
- Power Supply 5V

## REST API:
- GET/POST /api/config
- GET/POST /api/time
